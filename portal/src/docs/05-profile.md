# 个人中心

## 账号信息

查看和修改用户名、显示名称等个人信息。

## API Key 管理

API Key 是调用 AI 模型的凭证，**每人只能拥有一个活跃的 Key**：

- **创建** — 生成 Key 后直接可用，如已有 Key 则无法再次创建
- **重置** — 先清理旧 Key 再生成新 Key，旧 Key 立即失效
- **删除** — 删除后不可恢复，需重新创建

![API Key 管理列表](/docs/img/profile-apikey-list.png)

> ⚠️ **注意**：**每人只能拥有一个活跃的 API Key**。如已有 Key 需要更换，请使用「重置」功能，系统会先清理旧 Key 再生成新 Key。
