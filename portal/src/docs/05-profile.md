# 个人中心

## 账号信息

查看和修改用户名、显示名称等个人信息。

## API Key 管理

API Key 是调用 AI 模型的凭证，**每人只能拥有一个活跃的 Key**：

- **创建** — 生成 Key 后直接可用，如已有 Key 则无法再次创建
- **重置** — 先清理旧 Key 再生成新 Key，旧 Key 立即失效
- **删除** — 删除后不可恢复，需重新创建

![API Key 管理列表](/docs/img/profile-apikey-list.png)

> ⚠️ **注意**：**每人只能拥有一个活跃的 API Key**。如已有 Key 需要更换，请使用「重置」功能，系统会先清理旧 Key 再生成新 Key。

## 提交技能

个人中心提供「提交技能」入口，跳转到技能提交表单。该入口由管理员控制是否开放：

- 管理员开启后，按钮可见，可进入表单提交技能
- 未开放时按钮隐藏，普通用户无法提交

提交技能的详细流程见「提交技能」章节；上架后用户可用 `skillctl` 命令行安装，详见「skillctl 命令行说明」章节。

> 💡 **提示**：若看不到「提交技能」按钮，说明管理员当前未开放该功能，请联系管理员。
