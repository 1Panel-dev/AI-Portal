# Skill 市场

在 Skill 市场中你可以：

- 按分类（技能 / 技能包）、来源（本地 / 1Panel）筛选
- 按下载量、发布时间、评分排序
- 点击技能卡片查看详情和安装方式

![Skill 市场筛选栏 + 卡片网格](/docs/img/marketplace-grid.png)

## 来源说明

- **本地**：用户上传到本平台、经管理员审核通过的技能。
- **1Panel**：从 1Panel 企业版 Skills Hub 同步过来的官方技能。

下载、评分、详情等交互在两类来源上保持一致。
