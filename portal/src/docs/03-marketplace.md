# Skill 广场

在 Skill 广场中你可以：

- 按分类（技能 / 技能包）筛选
- 查看当前开放的 **1Panel** 来源技能
- 按下载量、发布时间、评分排序
- 点击技能卡片查看详情和安装方式

![Skill 广场筛选栏 + 卡片网格](/docs/img/marketplace-grid.png)

## 来源说明

当前仅开放 1Panel 来源：从 1Panel 企业版 Skills Hub 同步过来的官方技能。

本地来源暂不开放，后续版本再加入。
