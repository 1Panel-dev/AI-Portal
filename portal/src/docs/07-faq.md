# 常见问题

## 如何登录管理后台？

在登录页点击「管理员登录」，使用管理员账号登录。**普通用户无法访问管理后台。**

## 看不到「提交技能」按钮？

「提交技能」入口由管理员控制是否开放，默认关闭。若个人中心没有该按钮，说明管理员当前未开放用户提交，请联系管理员开启。

## 提交技能后一直没上架？

技能提交后需要管理员**审核通过**，通过后会自动上架展示。请耐心等待或联系管理员处理。上架后用户即可在 Skill 广场浏览，或通过 `skillctl install <skill-id>` 命令行安装（详见「skillctl 命令行说明」章节）。

## API Key 创建失败怎么办？

检查网络连接是否正常，稍后重试。如持续失败，请联系管理员检查系统配置。
