# 提交技能

> ⚠️ **注意**：「提交技能」入口由管理员开关控制。若你在个人中心看不到「提交技能」按钮，说明管理员当前未开放用户提交，请联系管理员开启。

## 1. 填写信息

在「提交技能」页面按表单逐项填写：

- **技能 ID**（必填）— 英文小写 + 连字符，例如 `my-awesome-skill`，作为技能唯一标识
- **显示名称**（必填）— 展示给用户看的中英文名称
- **描述**（必填）— 说明技能功能和使用场景
- **分类**（必填）— 当前为「技能」
- **版本号** — 遵循 semver 规范，如 `v1.0.0`；同 ID 同版本重复提交会覆盖之前待审核的提交，新版本请递增版本号
- **技能包**（必填）— `.zip` 格式，拖拽或点击选择上传

表单底部会根据技能 ID **自动生成安装命令**，提交上架后用户即可通过 `skillctl` 一键安装：

```bash
skillctl install <skill-id>
```

![提交技能表单](/docs/img/submit-form.png)

## 2. 等待审核

提交后技能包进入待审核状态，由管理员审核。通过后自动上架到 Skill 广场。

## 3. 查看状态

在「我的技能」页面查看审核进度（待审核 / 已通过 / 已拒绝）。

> 💡 **提示**：技能上架后，用户既能在 Skill 广场看到，也能用 `skillctl` 命令行直接安装。skillctl 的完整用法见左侧导航「skillctl 命令行说明」章节。
