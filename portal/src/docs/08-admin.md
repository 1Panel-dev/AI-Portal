# 管理员配置

本章节面向管理员，说明系统初始化与日常运营所需的配置。普通用户无需阅读。

## 登录管理后台

1. 访问站点 `/admin` 进入管理员登录页。
2. 使用管理员账号登录。首次启动的默认账号为 `admin`，默认密码 `admintest`（由 `.env` 中 `INIT_ADMIN_PASSWORD` 控制，生产部署前务必修改）。

> ⚠️ **注意**：普通用户账号无法登录管理后台。管理后台有独立的鉴权 token，与普通用户互不串扰。

登录后顶部导航提供五个入口：**审核管理、技能管理、用户管理、系统配置、第三方登录**。初始化时按下面顺序配置即可让站点可用。

## 系统配置

「系统配置」页面分为两个子标签：**1Panel 网关** 和 **站点设置**。这是初始化最关键的一步。

### 1Panel 网关

对接 1Panel 企业版 AI 网关，用于聚合模型列表、技能列表，并为用户创建 API Key。**不配置此项，模型广场与 API Key 功能均不可用。**

| 字段 | 说明 |
|------|------|
| 网关地址 (Base URL) | 1Panel 服务地址，需包含协议和端口，如 `http://192.168.1.100:33846` |
| API Key | 在 1Panel「设置 → API 接口」中创建；留空表示保留原值，填新值则覆盖 |
| 请求超时 (ms) | 调用 1Panel 的超时时间，默认 10000，最小 1000 |
| 同步间隔 (分钟) | 定时拉取模型/技能的间隔，最小 1，推荐 10 |

下方还有三个开关与一个角色选择：

- **启用定时同步** — 关闭后模型和技能列表不再自动从 1Panel 拉取（仍可手动「立即同步」）。
- **用户提交技能时同步上传到 1Panel** — 开启后，普通用户上传技能包会先写入 1Panel Skills Hub；上传失败则本次提交失败，不进入本地审核。关闭则用户提交仅进本地审核队列。
- **允许用户提交技能** — 控制普通用户能否看到「提交技能」入口。关闭后入口隐藏，后端提交接口也会拒绝。
- **1Panel 新建用户角色** — 通过本平台创建 1Panel 用户时使用的角色，角色列表来自 1Panel。保存后对新建用户生效。

配置流程建议：

1. 填写 Base URL 和 API Key。
2. 点击「测试连接」验证连通性。失败时常见原因：地址不通、API Key 无效、1Panel 端 IP 白名单未放行本机。
3. 测试通过后点击「立即同步」拉取一次模型和技能，确认广场有数据。
4. 点击「保存配置」持久化。页面下方「最近同步」会展示上次模型/技能同步的时间与结果。

> 💡 **提示**：若用户反馈"页面没数据"或"重置 API Key 失败"，先看后端日志里的 `[panel-call]` 行，能区分是网络问题、1Panel 业务码失败，还是配置缺失。

### 站点设置

「站点设置」子标签下分四块：调用示例、站点品牌、公告横幅、首次访问详细公告。

#### 调用示例

配置前台模型广场展示给用户的 Base URL 和 curl 示例模板。模板支持三个占位符：

| 占位符 | 含义 |
|--------|------|
| `{{base_url}}` | 调用地址，取自上方「调用地址」字段 |
| `{{model_name}}` | 模型名，由用户在模型广场复制 |
| `{{api_key}}` | 用户的 API Key |

保存后刷新前台模型页生效。

#### 站点品牌

| 字段 | 说明 |
|------|------|
| 站点名 | 显示在顶部导航栏 Logo 旁、浏览器标签页标题。最多 100 字符，不能为空 |
| 站点 Logo | 支持 PNG / JPG / SVG / WebP，最大 2 MB |
| 浏览器标签图标 (Favicon) | 建议 ICO 或 SVG，32×32 或 64×64 |

上传图片后需点击「保存」才生效。浏览器标签页标题会跟随站点名自动变化。

#### 公告横幅（顶部滚动条）

站点顶部蓝色滚动横幅。启用后填写横幅内容，支持 HTML：

- `<strong>` 加粗、`<a>` 链接、`<span class="banner-emphasis">` 黄色加粗强调
- 保存时 `<script>`、事件属性（如 `onclick`）、`javascript:` 链接会被自动剥离（防 XSS）

#### 首次访问详细公告（居中对话框）

访客首次进入站点时弹出的详细说明对话框。可配置标题与正文（HTML），正文额外支持：

- `<div class="dialog-alert">` — 黄色提示框
- `<p class="dialog-footnote">` — 小字脚注

> 💡 **提示**：保存任意变更后，所有曾点过「不再提示」的用户会在下次访问时重新看到此对话框（按公告版本号失效）。

## 第三方登录

「第三方登录」页面配置企业微信等第三方账号扫码登录。

### 启用前必读：可信域名与可信 IP

启用企业微信登录前，需在企业微信后台 → 应用管理 → 进入企业微信应用 → 「网页授权及 JS-SDK」中完成：

1. **可信域名** — 把当前部署域名加入，并完成域名所有权校验（下载 `WW_verify_xxx.txt` 放到根路径）。
2. **可信 IP** — 把服务器出口 IP 加入，保障回调请求来源可信。

页面会自动展示当前部署域名，可直接复制。

### 回调地址

页面自动生成 OAuth 回调地址（根据当前部署域名推导），无需手动填写。企业微信扫码完成后会回调到此地址。可一键复制填入企业微信后台。

### 与自助注册的联动

- **未启用任何第三方登录时**：自助注册接口正常开放，用户可自行注册账号。
- **启用任一第三方登录后**：自助注册接口自动关闭。新用户由管理员在「用户管理」中添加，或在首次扫码后通过「跳过绑定」自动创建。

> ⚠️ **注意**：启用第三方登录会改变用户来源方式，请提前规划账号体系。

## 用户管理

「用户管理」页面维护门户用户。

- **同步用户** — 从 1Panel 同步用户到本地。
- **新增用户** — 手动创建用户（角色取自「1Panel 新建用户角色」配置）。
- **批量改密** — 勾选多个用户后批量修改密码。
- **改密 / 删除** — 单个用户操作。删除时会先清理 1Panel 远端的用户与 API Key，再删除本地记录。

> ⚠️ **注意**：删除用户会联动清理 1Panel 远端资源，操作不可逆，请确认后再执行。

## 审核管理与技能管理

- **审核管理** — 处理用户提交的技能（通过 / 驳回）。通过后技能自动上架。
- **技能管理** — 维护已上架技能（下架、删除等）。涉及 1Panel Skills Hub 状态变更的操作会同步到远端。

## 配置速查

初始化最小可用配置清单：

1. `.env` 设置 `INIT_ADMIN_PASSWORD`（修改默认管理员密码）。
2. 管理后台「系统配置 → 1Panel 网关」填入 Base URL + API Key，测试连接并立即同步。
3. 「站点设置 → 站点品牌」设置站点名（默认即可，按需上传 Logo / Favicon）。
4. 「站点设置 → 调用示例」配置模型调用 Base URL 与 curl 模板，方便用户对接。
5. 按需在「第三方登录」启用企业微信登录。
6. 按需在「1Panel 网关」开启「允许用户提交技能」。
