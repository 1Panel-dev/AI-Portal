# Security policy

## Supported versions

| Version | Supported          |
| ------- | ------------------ |
| v0.x    | :white_check_mark: |

## Reporting a vulnerability

We first appreciate and are very thankful that you've found a vulnerability issue in AI-Portal. By disclosing such issue to the AI-Portal development team you are helping AI-Portal become a safer project.

To protect the existing users of AI-Portal, we kindly ask you not to disclose the vulnerability to anyone except the AI-Portal development team before a fix has been rolled out. Send an email to `wanghe@fit2cloud.com` instead.
